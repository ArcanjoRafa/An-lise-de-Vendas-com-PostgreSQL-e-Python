O projeto é um analise a um banco de dados que contem diversas tabelas focadas ao consumo de clientes em uma determinada loja de aparelhos eletrônicos. O foco aqui é mostrar como podemos analisar dados como os produtos mais vendidos,
os clientes mais fieis, o lucro obtido pela loja em meses alternados e muito mais. O projeto mostra desde a conexão com o PostgreSQL(Adminpg) até as analises finais mostradas em graficos como forma de facilitar a analise. Fiz este projeto
com foco em estagio com boas práticas, organização de código, segurança de credenciais e geração de insights a partir de dados relacionais.

->Tecnologias Utilizadas
  Python
  PostgreSQL
  SQL (JOIN, GROUP BY, ORDER BY, DATE_TRUNC)
  Pandas
  Matplotlib
  psycopg2
  python-dotenv
  Git
